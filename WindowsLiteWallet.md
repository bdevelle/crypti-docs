# Setting up The Crypti Lite Wallet on Your Windows Computer

----------

##1. Getting started 
**First you will need to download the Crypti Lite Wallet installer** 

[Crypti Lite](http://downloads.crypti.me/crypti-node/0.2.x/crypti-windows-lite-0.2.0.exe)  

**You will also need to download a NIST Internet Time Service (ITS) program**

[Atomic Clock Sync](http://www.worldtimeserver.com/atomic-clock/Download.aspx?t=ac)

##2. Installing and Customizing Your Crypti Lite Wallet 
- **Go to the folder you downloaded the Wallet to**

- **Double click and Run the file "crypti-windows-lite-0.2.0.exe"**

- **Follow the Setup Wizard** ( This will place an icon on your desktop )

- **You can now Finish and Launch the Wallet**

**You will see the below window**

  <a href="url"><img src="http://i.imgur.com/KX4Huh2.png" align="center" height="300" width="400" ></a>

*Note:* You will notice the **Default** setting is to **Automatically** find and **Connect** to the **Best** peer available.

**Once the Wallet has found a peer you will see window below**

<a href="url"><img src="http://i.imgur.com/AV8t6Qy.png" align="center" height="300" width="400" ></a>

###*Note:* At this point your Wallet is 100% Setup, if you would like to Customize it follow the steps below, otherwise continue on to Step 3
----------
####**To Customize This Wallet to Connect to either Your own Node or the Node you trust the most Follow the Steps below**

- **Click the Cloud in the Right upper corner**

<a href="url"><img src="http://i.imgur.com/HIgBgKp.png" align="center" height="300" width="400" ></a>

- **Uncheck the Box next to "Automatic peer"**

<a href="url"><img src="http://i.imgur.com/lkLxEXP.png" align="center" height="300" width="400" ></a>

- **Now enter the IP and Port of the peer you wish to Connect to** (Crypti 2.0 uses Port 8040)

<a href="url"><img src="http://i.imgur.com/ZcLq1pB.png" align="center" height="300" width="400" ></a>

- **Now click Save** (The Wallet will Now Ping the Node and make sure it's online)

<a href="url"><img src="http://i.imgur.com/AYHLiIj.png" align="center" height="300" width="400" ></a>

*Note:* The IP **255.255.255.255** is not a real Node IP so it returns an Error

- **Once you have entered in a Proper Node IP you will see the window below and be able to use the Wallet**

<a href="url"><img src="http://i.imgur.com/AV8t6Qy.png" align="center" height="300" width="400" ></a>

##3. Synchronizing your Local Windows clock to a World Time Server 

- **Go to the folder you downloaded "Atomic Clock Sync" to**

- **Double click and Run the file "Atomic.exe"**

**It will look something like this**

<a href="url"><img src="http://i.imgur.com/oZS8gN2.png" align="center" height="300" width="250" ></a>

**You can have these setup to automatically sync how ever often you would like** (There is also a manual Sync button)

<a href="url"><img src="http://i.imgur.com/J2oxOZU.png" align="center" height="300" width="250" ></a>

**If you run in to problems Follow the steps under the Repair tab**

<a href="url"><img src="http://i.imgur.com/IabkmYX.png" align="center" height="300" width="250" ></a>

###By this point you should have Installed the Wallet, Synchronizing your clock to Server Time and have a fully functioning Crypti Lite Wallet.
----------
